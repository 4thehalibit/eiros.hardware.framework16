# eiros.hardware.framework16

NixOS hardware configuration for the Framework Laptop 16. Part of the [Eiros](https://github.com/lcleveland/eiros) system configuration.

## Contents

- `hardware-configuration.nix` — generated hardware config with disk UUIDs and kernel modules
- `settings/monitors.nix` — monitor layout (built-in + two external ViewSonics)
- `flake.nix` — exposes this as a flake input
